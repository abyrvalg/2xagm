# 2xagm
In allows to monitor voltage on two 12V AGM accumulators connected in serial or parallel to avoid fall below degradation level of 10.5V

Used:
ESP32-WROOM-32E or ESP32-C3 SuperMini board
OLED 4 pin 128x32 0.91'' via I2C
two 1:16 deviders (6.8KOm and 102KOm) to ADC's

Arduino IDE 1.8.19

Drawing done using app.smartdraw.com
